# Atanas Patrick - Vue.js Portfolio

A modern, responsive portfolio website built with Vue.js 3, featuring advanced software engineering capabilities and stunning visual effects.

## 🚀 Features

### Core Technologies
- **Vue.js 3** with Composition API
- **Vite** for fast development and building
- **Tailwind CSS** for utility-first styling
- **Vue Router** for SPA navigation
- **Pinia** for state management
- **Vue Toastification** for notifications

### Advanced Features
- **🌙 Dark Mode Toggle** - Persistent theme switching
- **🔍 Project Search & Filtering** - Dynamic project discovery
- **✨ Advanced Animations** - Vue transitions and hover effects
- **📱 Fully Responsive** - Mobile-first design
- **🎨 Glassmorphism Effects** - Modern UI aesthetics
- **⚡ Performance Optimized** - Lazy loading and efficient rendering
- **📧 Contact Form Validation** - Real-time form feedback
- **🎯 Component Architecture** - Reusable Vue components

### UI/UX Features
- Gradient text effects and backgrounds
- Floating particle animations
- Interactive skill progress bars
- Project card hover effects
- Smooth page transitions
- Micro-interactions and animations
- Professional color scheme

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd portfolio-tailwind-main
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
